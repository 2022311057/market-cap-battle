# 時価総額バトル

日経225の主要企業（収録40社）のカードで対決する、ブラウザだけで遊べる株式知識カードゲームです。サーバー不要・依存ライブラリなしで、`index.html` を開くだけで動作します。

## 遊び方

1. ゲームを開始すると、収録企業（40社）のカードが7枚配られます。
2. 各ラウンド開始時に、為替・金利・需要動向などの**マクロイベント**がランダムに発生し、特定の業種の指標値を補正します。
3. **勝負指標**（時価総額・売上高・営業利益・従業員数）を選び、手札から1枚（またはM&Aで2枚合体）を選んで相手カードと対決します。
4. 5ラウンド終了後、合計スコアに応じたランクが表示されます。

## ルール概要

- **僅差ボーナス**: 差が15%未満の勝利は「絶妙！+30点」、差が5%未満の勝利は「ジャスト・キル！+50点」（基本点100点の最大50%まで）。
- **イベントカード**: 円安・AI特需などのイベントが売上高・営業利益を業種別に補正します（手札カードに補正値が色付きで表示されます）。
- **勝負指標の選択**: ラウンドごとに時価総額・売上高・営業利益・従業員数のいずれかで勝負できます。
- **M&Aシステム**: 手札に余裕がある場合、2枚のカードを合体して指標値を合算して対決できます。

## ファイル構成

```
index.html       画面構成（タイトル / ルール / ゲーム / 結果 / ゲームオーバー）
style.css        スタイル（単色アクセントのシンプルなデザイン）
js/data.js       企業データ・イベントデータ・指標定義
js/game.js       ゲームロジック（GameEngine、DOM非依存）
js/main.js       UI制御・DOM操作
```

## ローカルでの実行

ES Modulesを使わないグローバル構成のため、`index.html` をブラウザで直接開くだけで動作します。

```bash
# 任意：簡易サーバーで動作確認する場合
python -m http.server 8090
```

ブラウザで `http://localhost:8090/` または `index.html` を直接開いてください。
